# Prism API Schema

Prism is AdGem's GraphQL API for retrieving targeted offers. The pages in this
section document every query, type, input, enum, and scalar the API exposes.

- New to Prism? Start with the [Prism overview](/docs/reference/prism/overview).
- Need credentials? See [authentication](/docs/reference/prism/authentication).
- Want to try queries live? Use the [playground](/docs/reference/prism/playground).
