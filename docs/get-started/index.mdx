## Integration options

Choose the integration method that works best for your platform:

| Platform | Description | Best For |
|----------|-------------|----------|
| [**iOS SDK**](/docs/integrate/offer-delivery/pre-built/ios-sdk) | Native iOS SDK integration | iOS mobile apps |
| [**Android SDK**](/docs/integrate/offer-delivery/pre-built/android-sdk) | Native Android SDK integration | Android mobile apps |
| [**Unity SDK**](/docs/integrate/offer-delivery/pre-built/unity-sdk) | Cross-platform Unity plugin | Unity games |
| [**Web Offerwall**](/docs/integrate/offer-delivery/pre-built/web-offerwall) | JavaScript-based integration | Web applications |
| [**Offer API**](/docs/reference/offer-api/overview) | REST API for custom integrations | Custom implementations |
| [**Prism**](/docs/reference/prism/overview) | GraphQL API for targeted offers | Advanced targeting |
