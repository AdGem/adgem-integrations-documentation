# API Reference

The Reporting API gives you programmatic access to performance metrics for your AdGem applications. Use it to pull reporting data into your own dashboards, analytics pipelines, or internal tools.

## What You Can Do

- **Retrieve performance metrics** such as clicks, conversions, payouts, DAU, CTR, and eCPM
- **Group data** by app, date, platform, country, or offer
- **Filter results** by app ID, platform, or country
- **Customize response fields** to return only the data you need
