# Completion Difficulty Scale

The AdGem Completion Difficulty scale helps Offer API integrated Publishers gain more insight into offers. The scale (1-7) ranks offers from Easy to Hard as follows:

| Rank | Rank Parameters |
|------|-----------------|
| EASY (1) | All CPI, CPC, keyword campaigns have this ranking. Easy player registration or sign up (example: player inputs email, sets up a password; does not require external confirmation or personal information). 1-24 hours to complete offer. |
| MEDIUM - Non-Gaming (2) | Registration with personal information or external approval is needed in order to convert. Player initiates a trial start. 24-48 hours to complete offer. |
| MEDIUM - Gaming (3) | 24-48 hours of gameplay. |
| HARD - Non-Gaming (4) | Purchase campaigns. Subscription campaigns. 48-72 hours to complete offer. |
| HARD - Gaming (5) | 48-72 hours to complete offer. |
| EXTRA HARD - Gaming (6) | 48-72 hours to complete offer. |
| MULTI REWARD (7) | All multi-reward campaigns default to this ranking. Minimum 48-72 hours to complete offer. |
