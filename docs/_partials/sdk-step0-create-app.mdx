### Step 0: Create an App Property in AdGem

Before AdGem can populate offers, you need to create an App Property in the AdGem Dashboard.

1. Create a Publisher Account in the [AdGem Publisher Dashboard](https://dashboard.adgem.com/publisher/dashboard)
2. Register your App Property in [Properties & Apps](https://dashboard.adgem.com/publisher/apps)
