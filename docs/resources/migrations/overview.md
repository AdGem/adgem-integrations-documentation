# Migrations

Moving from an older AdGem integration to the current one? These guides cover what changed and how to switch. Each gives a side-by-side comparison and step-by-step instructions.

| Guide | From → To |
|---|---|
| [Static API → Offer API](/docs/resources/migrations/static-api-to-offer-api) | The legacy Static API (`GET /v1/all/campaigns`) → the [Offer API](/docs/reference/offer-api/overview) |
| [Postback v2 → v3](/docs/resources/migrations/postback-v2-to-v3) | GET-based v2 postbacks → signed, POST-based [v3 postbacks](/docs/integrate/reward-mechanism/postbacks-v3) |

The current integrations are recommended for new work — they're where new fields and improvements land. If you're unsure which you're on, or want help planning a migration, contact your Publisher Support Advocate.
