# Webhooks

AdGem sends server-to-server notifications (webhooks) to your servers to notify you about important events.

## Webhook Types

- **Postbacks** - Conversion notifications when users complete offers
- **Offer Events** - Status updates about offers
- **Banned Players** - Notifications when users are banned
