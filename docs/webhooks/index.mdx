# Webhooks & Postbacks

AdGem sends server-to-server notifications to your servers to keep you informed about important events in real time. These notifications fall into two categories: **postbacks** for conversion and install tracking, and **webhooks** for offer status changes and player suspension alerts.

## Postbacks

Postbacks notify you when players complete offers or install apps. They are essential for tracking conversions and rewarding players accurately.

- **[Postback Options](./postback-options.mdx)** - Overview of postback types, including install and reward/conversion postbacks
- **[Postback Setup](./offer-api-postback-setup.mdx)** - Step-by-step guide to configuring your postback URL, available macros, and security (IP whitelisting and postback hashing)

## Webhooks

Webhooks provide real-time notifications about offer-level changes and player suspension events.

- **[Offer Event Webhooks](./offer-event-webhooks.mdx)** - Real-time notifications when offers are created, updated, or removed (exclusive to API integrations)
- **[Suspended Player Webhooks](./suspended-player-webhooks.mdx)** - Instant alerts when players are suspended or unsuspended from AdGem
