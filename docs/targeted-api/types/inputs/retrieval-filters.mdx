The `RetrievalFilters` input type allows you to filter targeted offers when using the `getTargetedOffers` query. By providing specific filter criteria, you can narrow down the offers returned based on various dimensions such as categories, geographical regions, platform types, reward types, and tracking methods.

This input object is optional when calling `getTargetedOffers`, but using it enables more precise targeting to retrieve only the most relevant offers for your use case.


```graphql
input RetrievalFilters {
  categories: [String]
  country_code: String
  country_codes: String
  platform: String
  reward_type: String
  tracking_types: [String]
}
```
