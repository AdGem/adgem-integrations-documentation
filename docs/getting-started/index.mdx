## Integration Options

Choose the integration method that works best for your platform:

| Platform | Description | Best For |
|----------|-------------|----------|
| [**iOS SDK**](/docs/integrations/ios-sdk) | Native iOS SDK integration | iOS mobile apps |
| [**Android SDK**](/docs/integrations/android-sdk) | Native Android SDK integration | Android mobile apps |
| [**Unity SDK**](/docs/integrations/unity-sdk) | Cross-platform Unity plugin | Unity games |
| [**Web Offerwall**](/docs/integrations/web-offerwall) | JavaScript-based integration | Web applications |
| [**Offer API**](/docs/integrations/offer-api) | REST API for custom integrations | Custom implementations |
| [**Targeted API**](/docs/integrations/targeted-api) | GraphQL API for targeted offers | Advanced targeting |
